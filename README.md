# kotlin-minesweeper

## 지뢰 찾기

### 기능 요구 사항
- [x] 맵의 한 지점은 초기에 닫힌 상태이다.
- [x] 맵의 한 지점에는 닫힌 상태와 열린 상태가 있다.
- [x] 맵의 한 지점을 열면 열린 상태가 된다.
- [ ] 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- [ ] 지뢰는 눈에 잘 띄는 것으로 표기한다
- [ ] 지뢰는 랜덤에 가깝게 배치한다.
- [ ] 각 사각형에 표시될 숫자가 표시될 수 있다.
- [ ] 표시된 숫자는 자신을 제외한 주변 8개에 포함된 지뢰 수이다.
- [ ] 사용자가 open할 지점을 선택한다.
- [ ] 지뢰가 없는 인접한 칸이 모두 열린다.
- [ ] 지뢰를 밟으면 게임이 종료된다.
- [ ] 지뢰를 모두 찾으면 게임을 승리한다.
